Refs for 32 bit Windows versions:  Installation notes
=====================================================


1.  Uninstall any previous version.
==================================

If you have installed Refs for Windows NT/95 on this
computer and operating system before, uninstall it
before installing this version.

Your user settings will NOT be lost.

Windows 95:  Use the "Remove programs" applet in
             Control Panel

Windows NT:  run the 'Uninstall' icon in your 'Refs'
             group.


2. Install the program:
======================

Windows 95:  Use the 'Add/remove software' icon in
             control panel, and tell it the drive
             and directory the distribution files
             are in.

Windows NT:  Run the SETUP.EXE program in the
             distribution directory.
